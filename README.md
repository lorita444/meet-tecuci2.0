Proiectul MeetTecuci are ca scop o prezentare a orasului meu natal, pentru laboratorul de Tehnici Web de la facultate.

# STRUCTURA

# HTML
* `acasa.html`
  Pagina principală a aplicației, concepută pentru a capta atenția utilizatorului printr-un design modern și atractiv.

* `asezare.html`
  Conține informații despre poziționarea geografică și orientarea orașului.

* `contact.html`
  Permite utilizatorilor să trimită mesaje, sugestii sau reclamații prin intermediul unui formular interactiv, la care eu personal raspund cu interes.

* `istoric.html`
  Prezintă informații istorice despre Tecuci, îmbogățite cu elemente multimedia.

* `noutati.html` / `noutati_joc.html`
  Secțiuni dedicate actualizărilor recente, animațiilor și elementelor interactive (mini-jocuri sau experiențe dinamice).

# JavaScript

* `javra.js`
  Este fisierul principal de java, intuitiv nume. Gestionează logica principală a aplicației, inclusiv efecte dinamice și interacțiuni generale.

* `istoric.js`
  Controlează redarea fișierelor audio și animațiile din pagina de istoric.

* `reclamatie.js`
  Se ocupă de validarea formularelor din pagina de contact și de feedback-ul oferit utilizatorului după trimitere.

# CSS

* `stileto.css`
  Fisierul de stil care dă viața aplicației prin
    - culori vibrante, specifice tematicii și contrast plăcut
    - animații la hover (micro-interacțiuni)
    - un layout responsive pentru toate dispozitivele


## 🚀 Funcționalități principale

* Navigare intuitivă între pagini
* Design responsive (adaptabil pe mobil, tabletă și desktop)
* Elemente interactive și animații moderne
* Redare conținut multimedia
* Formular de contact cu validare în timp real
* Posibilitate de extindere pentru mai multe limbi

## Tehnologii utilizate

* HTML5 (structură semantică)
* CSS3 (design și animații)
* JavaScript (interactivitate și logică)
